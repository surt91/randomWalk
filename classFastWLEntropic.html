Detailed Description

Variant of Wang Landau sampling.

This variant does not need a flatness criterion for the histogram and will terminate after number of steps, determined by the $f$ parameter.

See http://arxiv.org/pdf/cond-mat/0701672.pdf

void FastWLEntropic::run ( )
virtual

Implementation of the "Fast" 1/t Wang Landau algorithm extended by Entropic Sampling.

Larger values of the final refinement parameter are ok, since the simulation will be "corrected" by an entropic sampling simulation after the Wang Landau estimation of g.

Literature used:

  • 10.1103/PhysRevE.75.046701 (original paper)
  • 10.1063/1.2803061 (analytical)
  • http://arxiv.org/pdf/1107.2951v1.pdf (entropic sampling)
