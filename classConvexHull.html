Detailed Description

template<class T>
class ConvexHull< T >

Class template for calculation of Convex Hulls.

Uses an algorithm to calculate the convex hull and some of its properties.

All properties are lazy evaluated.

Template Parameters
Tdatatype of the Steps (integer or double)
