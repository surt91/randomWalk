Detailed Description

Wang Landau Sampling of the distribution of a given observable.

Wang Landau sampling will sample the distribution over the full support. Takes usually longer than Metropolis sampling, but does not need parameters like temperatures or explicit equilibration.

The idea is to estimate a density function \(g\) on the fly and using this function to accept or reject changes as

\[ p_\mathrm{acc}(E_1 \to E_2) = \min\left(1, \frac{g(E_1)}{g(E_2)}\right). \]

The estimate of \(g\) is updated by multiplication with some refinement parameter \(f\), which is updated when \(g\) is a sufficiently good estimate of the real density function (i.e., the histogram of visited energies is sufficiently flat).

See http://arxiv.org/pdf/cond-mat/0011174.pdf
