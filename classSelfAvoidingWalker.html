Detailed Description

Self-Avoiding Random Walk.

A Walk which does not self intersect.

See also: doi: 10.1007/978-1-4614-6025-1_9 wiki

void SelfAvoidingWalker::change ( UniformRNG &  rng,
bool  update = true 
)
finalvirtual

Changes the walk, i.e., performs one trial move.

One change can consist of a "naive change" (80%) or a "pivot change" (20%). The "naive change" changes one step in the walk, such that all points after that are moved. The "pivot change" performs a symmetry operation at a random site in the walk, i.e., a rotation or mirroring. This leads to faster decorrelation but is seldom accepted.

Parameters
rngRandom number generator to draw the needed randomness from
updateShould the hull be updated after the change?
